🟢 Angular Material – Guia de Bolso
1️⃣ Cards
<mat-card class="mat-elevation-z4">
  <mat-card-title>Card Título</mat-card-title>
  <mat-card-content>
    <p>Conteúdo do card.</p>
  </mat-card-content>
  <mat-card-actions>
    <button mat-button>Cancelar</button>
    <button mat-raised-button color="primary">Confirmar</button>
  </mat-card-actions>
</mat-card>


mat-elevation-z4 → sombra

mat-card-title → título

mat-card-content → conteúdo

mat-card-actions → botões

2️⃣ Botões
<button mat-button>Simples</button>
<button mat-raised-button color="primary">Elevado</button>
<button mat-stroked-button color="accent">Contorno</button>
<button mat-flat-button color="warn">Flat</button>
<button mat-icon-button><mat-icon>favorite</mat-icon></button>


Tipos: mat-button, mat-raised-button, mat-stroked-button, mat-flat-button, mat-icon-button

Cores: primary | accent | warn

3️⃣ Inputs e Formulários
<mat-form-field appearance="fill">
  <mat-label>Nome</mat-label>
  <input matInput placeholder="Digite seu nome" [(ngModel)]="nome">
</mat-form-field>

<mat-form-field appearance="outline">
  <mat-label>Email</mat-label>
  <input matInput type="email" required>
  <mat-hint>Ex: usuario@dominio.com</mat-hint>
</mat-form-field>


appearance → fill | outline | standard

matInput → transforma <input> em Material

mat-hint → dica abaixo do campo

4️⃣ Toolbar
<mat-toolbar color="primary">
  <span>Meu App</span>
  <span class="spacer"></span>
  <button mat-button>Login</button>
</mat-toolbar>

<style>
  .spacer { flex: 1 1 auto; }
</style>


mat-toolbar → barra superior

color="primary" → cor do tema

5️⃣ Listas
<mat-list>
  <mat-list-item>
    <mat-icon mat-list-icon>person</mat-icon>
    <h4 mat-line>João</h4>
    <p mat-line>Cliente ativo</p>
  </mat-list-item>
  <mat-list-item>
    <mat-icon mat-list-icon>person</mat-icon>
    <h4 mat-line>Maria</h4>
    <p mat-line>Cliente inativo</p>
  </mat-list-item>
</mat-list>


mat-list → lista

mat-list-item → item da lista

mat-line → texto principal/secundário

6️⃣ Tabelas (Resumo)
<table mat-table [dataSource]="clientes" class="mat-elevation-z8">
  <ng-container matColumnDef="nome">
    <th mat-header-cell *matHeaderCellDef>Nome</th>
    <td mat-cell *matCellDef="let cliente">{{cliente.nome}}</td>
  </ng-container>

  <ng-container matColumnDef="email">
    <th mat-header-cell *matHeaderCellDef>Email</th>
    <td mat-cell *matCellDef="let cliente">{{cliente.email}}</td>
  </ng-container>

  <tr mat-header-row *matHeaderRowDef="['nome','email']"></tr>
  <tr mat-row *matRowDef="let row; columns: ['nome','email'];"></tr>
</table>


mat-table → tabela Material

mat-header-cell → cabeçalho

mat-cell → célula

mat-elevation-z8 → sombra

7️⃣ Dicas Visuais

mat-elevation-z<number> → sombra do componente

color="primary|accent|warn" → tema

appearance="fill|outline|standard" → inputs
